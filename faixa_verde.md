## 3. ROLAMENTOS

- **Ushiro hantem ukemi** = meio rolamento para trás  
- **Ushiro kaiten ukemi** = queda rolando para trás  
- **Mae ukemi** = queda para frente  
- **Mae kaiten ukemi** = queda rolando para frente  
- **Mae yoko ukemi** = queda lateral para frente  
- **Shukko** = andar de joelhos  

## Posturas

- **Seiza** = sentado sobre as pernas  
- **Zazen** = sentado com as pernas cruzadas  
- **Tachi** = em pé  
- **Suwari** = ajoelhado  
- **Mae** = frente  
- **Ushiro** = atrás  
- **Waza** = técnica  

## Fundamentos

- **1º Shisei** = Postura — Triângulo  
- **2º Sabaki** = Esquiva — Círculo  
- **3º Waza** = Técnica — Quadrado  

## 4. JŌ / KEN

- **Shōmen uchi gedan gaeshi**  
- **Shōmen uchi ushiro tsuki**  
- **Gyaku yokomen uchi ushiro tsuki**  
- **Katate gedan gaeshi**  
- **Katate toma uchi**
