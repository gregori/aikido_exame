## 1. VOCABULÁRIO / DEMONSTRAÇÕES

- **Kata sodedori** = aprisionamento na manga  
- **Katate dori** = aprisionar um braço  
- **Ryō katate dori** = segurar os dois braços do uke  
- **Ryō kata sodetori** = segurar as duas mangas do uke  
- **Kata munedori** = segurar a gola do uke  
- **Ryō kata munedori** = segurar as duas golas do uke  
- **Ushiro kata eri dori** = segurar na gola atrás do pescoço  
- **Ushiro tekubi dori** = um braço estrangula e o outro segura o pulso  
- **Kata dori men uchi** = segurar na manga e atacar a cabeça  
- **Kata dori yoko men uchi** = segurar na manga e atacar na lateral da cabeça  
- **Kata dori tsuki uchi** = segurar a manga e atacar com soco  
- **Katate dori men, yoko, tsuki uchi** = variações de ataque com uma mão segurando e a outra atacando  
- **Chūdan / Gedan / Jōdan tsuki** = soco médio / baixo / alto  
- **Yokomen uchi** = ataque na lateral da cabeça  
- **Shōmen uchi** = ataque no meio da cabeça  
- **Mae geri** = chute frontal  
- **Mawashi geri** = chute lateral

## 3. JŌ NO SUBURI

- **Hassō gaeshi uchi**  = base hassō com corte descendente
- **Hassō gaeshi tsuki**  = base hassō com estocada
- **Hassō gaeshi ushiro tsuki**  = base hassō com estocada para trás
- **Hassō gaeshi ushiro uchi**  = base hassō com corte para trás
- **Hassō gaeshi ushiro barai**  = base hassō com varredura para trás
- **Hidari nagare yokomen uchi**  = fluxo esquerdo com corte lateral
- **Migi nagare gaeshi tsuki**  = fluxo direito com estocada de retorno
- Esquiva com três ukes atacando shōmen e tsuki  
- Realizar **shukko tenkan**
